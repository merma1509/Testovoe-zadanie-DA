# Задача 2: Кулинарное соревнование

## Условие

В конкурсе участвуют 80 шеф-поваров с уникальными уровнями мастерства. В первом этапе судьи случайным образом распределяют их по парам (в любом состязании двух шефов выигрывает тот, у кого выше уровень мастерства). На втором этапе шефы снова случайно образуют пары для финального раунда (пары могут повториться). Победную награду получают те, кто выиграл в обоих этапах.

Каково математическое ожидание числа победителей?

## Решение

### Анализ ситуации

- 80 шеф-поваров с уникальными уровнями мастерства
- Каждый этап: случайное разбиение на 40 пар
- Победитель в паре: всегда шеф с более высоким уровнем мастерства
- Финальный победитель: должен выиграть оба этапа

### Вероятность победы для одного шеф-повара

Рассмотрим произвольного шеф-повара. Для него выиграть оба этапа:

**Этап 1:**

- Вероятность попасть в любую пару: 1 (все участвуют)
- Вероятность выиграть свою пару: зависит от уровня мастерства

**Этап 2:**

- Аналогично этапу 1
- Пары формируются независимо

### Ключевое наблюдение

Из-за уникальности уровней мастерства и случайности формирования пар:

Для любого шеф-повара вероятность выиграть одну случайную пару = 0.5

**Обоснование:**

- В любой паре всегда есть ровно один победитель и один проигравший
- При случайном формировании пар, для конкретного шефа:
  - С вероятностью 0.5 он встретит шефа с более высоким уровнем (проиграет)
  - С вероятностью 0.5 он встретит шефа с более низким уровнем (выиграет)

### Математическое ожидание

Пусть $X$ — количество победителей (выигравших оба этапа).

Для каждого шефа $i$ введем индикаторную переменную:

- $X_i = 1$, если шеф $i$ выиграл оба этапа
- $X_i = 0$, если шеф $i$ не выиграл оба этапа

Тогда: $X = \sum_{i=1}^{80} X_i$

По линейности математического ожидания:
$$E[X] = \sum_{i=1}^{80} E[X_i]$$

Для любого шефа:
$$P(\text{выиграть оба этапа}) = P(\text{выиграть этап 1}) \times P(\text{выиграть этап 2}) = 0.5 \times 0.5 = 0.25$$

Следовательно:
$$E[X_i] = 0.25$$

Итоговое математическое ожидание:
$$E[X] = 80 \times 0.25 = 20$$

## Ответ: **20,0**

## Проверка через симуляцию

Для проверки можно провести мысленный эксперимент:

- В каждом этапе выигрывает ровно 40 шефов (половина)
- Во втором этапе из 80 шефов снова выигрывают 40
- Математическое ожидание пересечения двух случайных подмножеств по 40 элементов из 80 = 20

## Интуитивное объяснение

Ожидается 20 победителей, что составляет 25% от всех участников. Это логично, так как:

- Каждый этап отсеивает половину участников
- Два независимых этапа дают вероятность 0.25 для двойной победы
- Из 80 участников ожидается 0.25 × 80 = 20 двойных победителей
